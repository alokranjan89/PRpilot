# PRPilot AI

An AI-powered GitHub mentor that helps beginners understand repositories, issues, and open-source projects.

## Tech Stack

### Backend
- Node.js
- Express.js
- TypeScript
- Prisma
- PostgreSQL
- Zod

### Current Features

- Express setup
- PostgreSQL with Prisma
- User CRUD
- Validation with Zod
- Global Error Handler
- Async Handler
